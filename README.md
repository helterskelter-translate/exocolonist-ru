## Инструкция по установке

1. Перейдите в корневую папку игры Exocolonist
Пример расположения игры: \SteamLibrary\steamapps\common\Exocolonist\
2. Распакуйте содержимое папки Exocolonist в папку игры с заменой файлов
3. Игра готова в запуску, приятной игры <3
