# Полный перевод игры I Was a Teenage Exocolonist на русский язык

Важно: перевод уже готов для игры, однако он всё ещё находится на стадии вычитки, возможны незначительные ошибки, которые будут исправлены в последующих версиях.
Текущий перевод оптимизирован только для мужского персонажа, работа над оптимизацией текста для обоих полов еще идет, учитывайте это во время игры.
Следите за обновлениями на нашем сайте, новая версия будет появляться каждую неделю вплоть до финальной редакции.

# Инструкция по установке

1. Перейдите в корневую папку игры Exocolonist
Пример расположения игры: \SteamLibrary\steamapps\common\Exocolonist\
2. Распакуйте содержимое папки Exocolonist в папку игры с заменой файлов
3. Игра готова в запуску, приятной игры <3
